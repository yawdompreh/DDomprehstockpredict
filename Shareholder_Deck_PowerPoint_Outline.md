# MarketPilot AI - PowerPoint-Ready Shareholder Deck Outline

Use this document as the direct build spec for a PowerPoint file. Each section maps to exactly one slide.

## Theme Guide (Apply Across All Slides)
- Aspect ratio: 16:9 widescreen
- Fonts: Headings = Aptos Display Bold, Body = Aptos Regular
- Color palette:
  - Ink: #112428
  - Teal: #0E8F7B
  - Burnt Orange: #D3542A
  - Sand: #F3EFE7
  - White: #FFFFFF
- Visual style: clean institutional layout with warm-tech accents
- Icon style: simple line icons, consistent stroke weight
- Motion (optional): Fade on section headers, Wipe for timeline/process visuals
